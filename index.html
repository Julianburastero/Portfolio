Experiencia

Análisis Taxis de NYC

Desarrollé el análisis de datos y un dashboard interactivo para optimizar el servicio de taxis en Nueva York, analizando facturación, demanda, geolocalización de viajes y emisiones de CO₂. Utilicé Power BI para las visualizaciones y Google Cloud con BigQuery para cargar los datos. Presenté insights sobre la actividad de los taxis y propuse la incorporación de vehículos eléctricos. El sistema de recomendación fue implementado en Render y Streamlit.

Análisis Internet Argentina

Realicé un análisis exhaustivo del uso de Internet en Argentina, desde la extracción y limpieza de datos hasta su visualización en un dashboard interactivo. Implementé Python (Pandas, NumPy) para el proceso ETL, Seaborn y Matplotlib para el análisis exploratorio, y Power BI con DAX para la creación de métricas clave. Identifiqué patrones, tendencias y outliers, facilitando la toma de decisiones sobre la conectividad en distintas provincias.

API Recomendación Películas - MVP

Desarrollé un Sistema de Recomendación de Películas utilizando Machine Learning y Procesamiento de Lenguaje Natural (NLP). Implementé un modelo basado en la similitud del coseno para sugerir películas según el título ingresado por el usuario. Además, la API ofrece funcionalidades como estrenos por mes o día, puntuaciones basadas en críticas, análisis de actores y directores, y recomendaciones de películas similares. Utilicé Python, FastAPI, Scikit-learn, Pandas y NLTK, desplegando la API en Render. El proceso incluyó ETL, EDA y el desarrollo del modelo.
